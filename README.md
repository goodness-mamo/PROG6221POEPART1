# PROG6221POEPART1
Cybersecurity Awareness Bot
Overview

The Cybersecurity Awareness Bot is a C# console application designed to educate users about essential cybersecurity practices through an interactive chatbot experience.

The bot allows users to ask questions and receive guidance on topics such as:

Password security 
Phishing attacks 
Safe browsing 

It also includes enhanced user experience features such as a voice greeting, ASCII art interface, and a typing animation.

Objectives
Promote cybersecurity awareness
Provide simple and practical safety tips
Demonstrate core C# programming concepts
Create an interactive console-based chatbot
 Features
 Voice Greeting using .wav file
 ASCII Art UI for visual appeal
 Interactive Chatbot with continuous conversation
 Typing Animation Effect for realism
 Keyword-Based Responses for cybersecurity topics
 Error Handling for missing files
